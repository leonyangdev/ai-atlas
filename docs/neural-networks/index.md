# 神经网络架构

> 从最简单的前馈网络到能处理序列的循环网络，再到能翻译语言的 Seq2Seq——这里记录了各类神经网络架构的原理与实现。

## 架构演进路线

```
FNN（前馈神经网络）  — 最基础的结构，处理定长向量
    ↓
RNN（循环神经网络）  — 引入时间步，处理序列数据
    ↓
LSTM                — 门控机制，解决 RNN 的长距离遗忘
    ↓
GRU                 — 简化版 LSTM，效果相当但更高效
    ↓
Seq2Seq             — 编码器-解码器架构，用于机器翻译
    ↓（下一章）
Attention + Transformer — 并行注意力机制，取代 RNN
```

## 本章目录

| 架构 | 核心问题 | 主要应用 |
|---|---|---|
| [FNN（前馈网络）](/neural-networks/fnn) | 定长向量 → 分类/回归 | 表格数据，图像展平分类 |
| [RNN（循环网络）](/neural-networks/rnn) | 序列 → 序列/分类 | 文本分类，时间序列 |
| [LSTM](/neural-networks/lstm) | 长距离依赖记忆 | 机器翻译，语音识别 |
| [GRU](/neural-networks/gru) | 简化门控，高效建模 | 各类序列任务 |
| [Seq2Seq](/neural-networks/seq2seq) | 变长输入→变长输出 | 机器翻译，文本摘要 |

## 各架构核心对比

| 架构 | 输入 | 记忆能力 | 并行性 | 适用序列长度 |
|---|---|---|---|---|
| **FNN** | 定长向量 | 无 | ✅ | 不适用 |
| **RNN** | 任意长序列 | 短期 | ❌（串行） | 短序列（<50步） |
| **LSTM** | 任意长序列 | **长期** | ❌（串行） | 中等序列（<200步） |
| **GRU** | 任意长序列 | 长期 | ❌（串行） | 中等序列 |
| **Transformer** | 任意长序列 | **全局** | ✅（并行） | 长序列（支持数千步） |
